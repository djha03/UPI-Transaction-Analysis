# UPI Transaction Analysis Dashboard

An interactive Tableau dashboard built to analyze 20,000+ UPI transaction records across key business dimensions such as city, age group, payment method, merchant, device type, currency, and transaction status.

## Project Highlights
- City-wise transaction distribution using a geographic map
- Age-group-based customer transaction analysis
- Payment method and merchant performance comparison
- Success vs. failed transaction analysis
- Interactive filters for deeper drill-down insights

## Tools Used
- Tableau
- SQL

## Objective
The objective of this project is to uncover transaction trends, customer behavior, and business insights from UPI payment data to support data-driven decision-making.

## Key Insights
- Identified high-activity cities based on transaction volume and value
- Analyzed the most active customer age groups
- Compared transaction patterns across payment methods and merchants
- Evaluated transaction success and failure trends

## Dashboard Features
- Interactive maps and charts
- KPI-based analysis
- Dynamic filters by city, currency, device type, and status
- Clean and business-friendly layout

## Conclusion
This project provides a comprehensive view of UPI transaction behavior and demonstrates how Tableau can be used to transform raw transaction data into actionable business insights.
